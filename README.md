# tree-view

## Тестовое задание реализация Иерархического дерева

Реализовано:

* Отображение
* CRUD
* Поиск по списку
* Сортировка
* Совмещение сортировки и поиска

### Установка
```
npm install
```

### Компиляция and hot-reloads for development
```
npm run serve
```

### Компиляция and минификация for production
```
npm run build
```

### Линтинг and fixes files
```
npm run lint
```
